# 既知の問題

## 翻訳品質

### 品質ポリシー
本翻訳は厳密な正確性を目的としていません。インディーゲームをストレスなく遊べる範囲を目指しており、機械翻訳ベース＋人手調整で短期間に仕上げているため、誤訳・口調揺れ・意訳が含まれます。

### 一部の長文が意訳・短縮されている
バフ説明・敵説明・スキル説明の一部は、表示領域・byte 予算の都合で意訳されています。実機で「説明文の意味が通じにくい」「文脈と合わない」と感じた場合はご報告ください。

### キャラクター口調の一貫性
全 86 章を機械的かつ短期間で翻訳したため、章をまたいで同一キャラの口調が微妙に揺れる可能性があります。

### DLL UI に改行候補由来の半角スペースが残る
DLL から呼ばれる長文 UI（説明文など）には改行ロジックが適用され、改行位置が半角スペースに変換されます。結果として `手動 セーブ` のような不自然な空白が残ることがあります。実機表示を見ながら調整予定。

## 動作

### ゲーム言語設定
ゲーム内設定で言語を "中文" に切り替えると、日本語訳は表示されません（中国語スロットには注入していないため）。**English のままで運用**してください。

### Steam クラウド
ゲーム開始時に Steam クラウドが「クラウド上のオリジナルデータ」で上書きしてくるケースがあります。
* インストール直前にゲームをきちんと終了
* それでも上書きされる場合は、Steam のクラウド同期を当該タイトルで一時無効化

### ゲームアップデート
開発元がゲームを更新すると、`resources.assets` の構造が変わり本パッチが動作しなくなる可能性があります。アップデート後は uninstall.bat で復元してから、新バージョンの本パッチを待ってください。

## 既知の未翻訳箇所

### 内部ディレクティブ・固有名
* 内部ディレクティブ (`#动画切换`, `#隐藏`, `#留下金摇杆` 等) — ゲームロジックで参照されるため意図的に未訳
* `BGM` / `SE`（音量ラベル）— 略号のまま
* CN/EN いずれも同じテキスト（例: 数字・特殊記号）
* キャラクター固有名（允星 / 浮昔 / 海豹 / 舔狗 / 言隐 / 豚可 / 咕咕 / 咕咕群主 等）は中国語原文のまま保持

### チートメニュー (cheater mode) の Buff / Monster プルダウン項目
チートメニューの **Buff プルダウン** と **モンスター プルダウン** の項目は中国語表示のまま残ります。

例:
- Buff: 加速 / 击退 / 满血 / 杀怪加钱 / 无敌 / 中毒 / 冻结 / 击晕 等
- モンスター: 史莱姆球 / 小史莱姆 / 爆炸史莱姆球 / 大史莱姆 / 巨大史莱姆 / 水母 / 大水母 / 人鱼 / 章鱼 / 海葵 等

#### 理由
これらの CN 文字列は `resources.assets` 内に多数（buff 1種あたり 16〜19 出現位置）保存されており、ゲームの C# コードが状態判定や型分岐のキー（例: `if buff.name == "无敌"`）として直接参照しています。

実験的にこれら CN 文字列を日本語に置換した結果、**ゲームが起動しなくなる現象**を確認しました（Unity ローダーが MonoBehaviour シリアライズ層で停止）。**ゲームの安定動作を優先**するため、CN 表記を維持する方針としました。

#### 日本語化済み（チートメニューで日本語表示される項目）
- パネル本体のラベル: 最大HP / HP / HP固定 / 無敵 / 死亡回数 / モンスター / モンスター追加 / 味方スライム追加 / Buff追加 / スキル解放 / 全体ダメージ / 全体即殺 / STEAM実績リセット
- 「現在の状態」のドロップダウン値: 戦闘前 / 戦闘中 / 戦闘後
- バフ・敵・スキルの **説明文** （図鑑表示）は日本語化（プルダウンの項目名のみ CN）
